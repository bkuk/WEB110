Creating a Basic HTML File

The first step is to create an html file. This is very simple, just open up a text editor and save it as something.html. This will tell the computer that it is saved as an HTML file. I included this step because it was not very obvious to me how to start the web development process and because step one is the most important.

We then need to tell the file that we are going to be writing in HTML. We can do this by including a "!DOCTYPE html" in angle brackets at the top of the file. This is basically telling the file that HTML is going to be written in the file.
